# 🎶 Guia de sobrevivencia: dominio do terminal

este repositorio foi criado como parte da disciplina de **sistemas operacionais**, utilizando o simulador [Terminal Master]()
para praticar comandos essenciais de windows e linux.

---

## 👽 Introduçao 
A interface de linha de comando (CLI) é uma ferramenta poderosa que permite interagir com o nucleo do sistema operacional. Dominar
esses comandos é fundamental para automação de tarefas, admisnistração de servidores e desenvolvedores e desenvolvimento software.

---

## 🛹 Comandos Aprendidos 

Abaixo, apresento a lista dos principais comandos praticados durante o treinamento gamificado:

| Comando | Função | Sistema | Exemplo de uso |
| :--- | :--- | :--- | :--- |
| `pwd` | Exibe o diretório atual | Linux | pwd |
| `cd` | Altera o diretório de trabalho | Linux/Windows | cd Documentos |
| `ls` | Lista arquivos e pastas | Linux | ls |
| `dir` | Lista arquivos e pastas | Windows | dir |
| `mkdir` | Cria uma nova pasta | Linux/Windows | mkdir nova_pasta |
| `rmdir` | Remove pasta vazia | Linux/Windows | rmdir pasta |
| `rm` | Remove arquivos ou pastas | Linux | rm arquivo.txt |
| `del` | Remove arquivos | Windows | del arquivo.txt |
| `cp` | Copia arquivos e pastas | Linux | cp a.txt b.txt |
| `copy` | Copia arquivos | Windows | copy a.txt b.txt |
| `mv` | Move ou renomeia arquivos | Linux | mv a.txt pasta/ |
| `move` | Move arquivos | Windows | move a.txt pasta\ |
| `rename` | Renomeia arquivos | Windows | rename a.txt b.txt |
| `clear` | Limpa o terminal | Linux | clear |
| `cls` | Limpa o terminal | Windows | cls |
| `touch` | Cria um arquivo vazio | Linux | touch arquivo.txt |
| `echo` | Exibe texto na tela | Linux/Windows | echo Olá |
| `cat` | Mostra conteúdo do arquivo | Linux | cat arquivo.txt 
| `type` | Mostra conteúdo do arquivo | Windows | type arquivo.txt |
| `find` | Procura arquivos | Linux | find . -name teste.txt |
| `where` | Localiza arquivos | Windows | where notepad |
| `chmod` | Altera permissões de arquivos | Linux | chmod 777 arquivo |
| `chown` | Altera o dono do arquivo | Linux | chown user arquivo |
| `ping` | Testa conexão de rede | Linux/Windows | ping google.com |
| `ipconfig` | Mostra o IP do computador | Windows | ipconfig |
